# 02. レイアウトの基本（Grid / StackPanel）

## 画面構成の考え方

WPF の画面は次の2段構えで考えます。

- Grid で画面を分割する
- Panel で要素を並べる

---

## サンプル構成

- 左ペイン：縦方向 StackPanel
- 右ペイン：横方向 StackPanel
- GridSplitter により左右サイズ変更可能

---

## Grid の列定義

- 固定幅
- *
- スプリッター

---

## StackPanel の Orientation

Vertical / Horizontal の違い
