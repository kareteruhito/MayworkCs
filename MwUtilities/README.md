# MwUtilities

MwUtilities は、**UI やフレームワークに依存しない static Utility メソッド群**をまとめたプロジェクトです。

このプロジェクトは **ライブラリ配布を目的としていません**。  
各 Utility は、必要に応じて **`.cs` ファイルをコピー＆ペーストして利用する**ことを想定しています。

---

## このプロジェクトの目的

- UI 非依存の **汎用的な処理・計算ロジック**の置き場
- 小さな処理を試すための **即席動作確認環境**
- 実アプリで使いながら Utility をブラッシュアップし、  
  改良内容をこのプロジェクトへ還元する

**「使い捨てにならないコード片を育てる場所」**を目的としています。

---

## プロジェクト形態について

- **Console プロジェクト**
- WPF / WinForms などの UI 参照なし
- `System.*` を中心とした最小依存

Console プロジェクトにすることで、

- すぐ実行して結果を確認できる
- 単体テストを書くほどでもない処理を気軽に検証できる

というメリットがあります。

---

## 基本方針

- すべて **static メソッド**
- 状態を持たない（または極力持たない）
- 入力 → 出力が明確
- アプリケーションロジックは含めない
- フレームワーク依存を持ち込まない
